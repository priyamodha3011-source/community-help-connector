# Community Help Connector

Community Help Connector is an AngularJS-based web application that connects people who need help with volunteers. Users can sign up, submit help requests, register as volunteers, and manage community support efficiently.

## Features
- User Sign Up / Login
- Submit Help Requests
- Volunteer Registration
- Accept / Reject Requests
- Simple, professional and attractive interface

## Technologies Used
- HTML
- CSS
- JavaScript
- AngularJS
